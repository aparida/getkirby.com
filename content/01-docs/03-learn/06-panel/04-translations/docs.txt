Any help with translations in other languages is very welcome!!
You can fork Kirby on Github or download the (link: https://github.com/bastianallgeier/kirbycms-panel/blob/dev/languages/en.php text: english language file) as reference and just translate it. Just send me an email if you got any questions: <bastian@getkirby.com>
