Title: html::a
----
Syntax: html::a($href, $text = null, $attr = array())
----
Description: Generates an a tag
----
Text: 
----
Parameters: - name: $href
  type: string
  text: The url for the a tag

- name: $text
  type: mixed
  text: The optional text. If null, the url will be used as text

- name: $attr
  type: array
  text: Additional attributes for the tag
----
Returntype: string
----
Return: the generated html
