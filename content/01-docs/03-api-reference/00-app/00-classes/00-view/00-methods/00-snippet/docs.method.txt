----
Parameters: - name: $path
  type: string
  text: The smart path to the snippet

- name: $data
  type: array
  text: Optional data, which should be passed to the snippet

- name: $return
  type: boolean
  text: By default the snippet is echoed, but you can return the result by passing true here.
