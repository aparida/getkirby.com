Title: plugin::load
----
Syntax: plugin::load($file)
----
Description: Loads an additional sub file of the plugin
Use it like this $this->load('lib' . DS . 'somefile.php')
to load somefile.php within the lib subfolder of the plugin folder
----
Text: 
----
Parameters: - name: $file
  type: string
  text: The relative path to the loadable file. This can also be an array of files.
