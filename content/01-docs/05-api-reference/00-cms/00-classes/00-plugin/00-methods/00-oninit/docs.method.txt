Name: onInit
----
Visibility: public
----
Static: no
----
Title: plugin::onInit
----
Syntax: plugin::onInit($arguments = array())
----
Description: Is being triggered when the instance method
is called for the first time. Afterwards the
instance is cached and this won't be called again.
----
Text: Use this in child classes to create an instance of
any object you want to attach to the plugin
----
Parameters: - name: $arguments
  type: array
  text: Optional arguments to pass to the instance
----
Returntype: false
----
